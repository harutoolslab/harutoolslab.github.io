  • Esta ferramenta é um simulador simplificado que corre no navegador. Devido à natureza do programa, não podemos excluir totalmente a possibilidade de comportamentos inesperados ou bugs em condições específicas.

O que é o Simulador de Circuitos Lógicos?

Esta é uma ferramenta online que permite construir e testar circuitos lógicos digitais de forma intuitiva no seu navegador. Suporta desde portas lógicas básicas (AND, OR, NOT, etc.) até flip-flops essenciais para circuitos de memória. É ideal para trabalhos escolares, preparação para eletrónica ou como ferramenta de aprendizagem para aprofundar a compreensão de circuitos digitais.

Instruções Detalhadas de Operação

  • Colocação e Movimento: Arraste os botões do painel superior esquerdo ou clique neles para adicionar componentes. Pode mover os blocos livremente clicando e arrastando-os.
  • Cablagem: Clique e arraste a partir dos terminais "●" nos lados dos componentes para os ligar a outros terminais.
  • Alterar Entradas: Pressione a tecla Shift enquanto clica num SWITCH para alternar o estado LIGADO/DESLIGADO.
  • Eliminar: Clique com o botão direito do rato em componentes ou fios para os remover individualmente.
  • Visualização: Arraste com o botão direito no fundo para mover o cenário e use a roda do rato para fazer Zoom (Aumentar/Diminuir).
  • Modo Ecrã Inteiro: Clique no botão "ECRÃ INTEIRO" no canto inferior direito para expandir o simulador.

Características desta Ferramenta

  • Sem Instalação: Funciona apenas no navegador, acessível a partir de qualquer sistema operativo.
  • Totalmente Gratuito: Todas as funcionalidades estão disponíveis sem custos ou registos. Pode guardar e carregar os seus projetos.
  • Simulação em Tempo Real: O comportamento é refletido no instante em que monta o circuito, com o estado da corrente visível através das cores.
  • Privacidade Garantida: Todo o processamento é feito localmente no seu navegador; os dados do seu circuito nunca são enviados para o servidor.

Passos Básicos para Utilização

  1. Escolha os componentes (como SWITCH ou portas AND) no painel "E/S / Portas" e coloque-os no centro do ecrã.
  2. Ligue os componentes com fios e verifique se a lógica pretendida está a funcionar.
  3. Após confirmar o funcionamento, pode clicar em "GUARDAR" para descarregar um ficheiro JSON.
  4. Pode retomar o seu trabalho a qualquer momento utilizando o botão "ABRIR".

Perguntas Frequentes (FAQ)

P. O simulador suporta circuitos complexos?

R. Circuitos combinatórios e sequenciais básicos (flip-flops, etc.) funcionam sem problemas. No entanto, circuitos extremamente grandes podem depender da velocidade de processamento do seu navegador.

P. Não consigo guardar o meu circuito.

R. Ao clicar em "GUARDAR", o ficheiro é descarregado automaticamente. Verifique as definições de permissão de descarga do seu navegador.

P. Posso utilizar em dispositivos móveis?

R. A interface foi desenhada para utilização com rato no PC. Embora seja possível visualizar em telemóveis, recomendamos um computador para operações precisas como cablagem e movimento.
