Cómo usar la herramienta de texto en imágenes

Esta herramienta te permite añadir texto a tus imágenes de forma sencilla directamente desde tu navegador. Ideal para crear banners, añadir notas a fotos para redes sociales, poner fechas en fotos de recuerdos y mucho más.

Funciones principales

  • Operación intuitiva: Mueve el texto añadido a cualquier lugar simplemente arrastrándolo con el ratón o el dedo.
  • Múltiples textos: Puedes colocar tantos textos diferentes como quieras en una sola imagen.
  • Privacidad segura: Las imágenes no se suben al servidor. Todo el procesamiento se realiza en tu dispositivo, por lo que puedes editar fotos familiares o de trabajo con total tranquilidad.

Preguntas frecuentes

P. ¿Cómo puedo borrar un texto?

R. En la versión actual, deberás recargar la página o volver a subir la imagen para empezar de nuevo. Tenemos planeado añadir una función de borrado individual próximamente.

P. ¿Puedo usar diferentes fuentes?

R. Sí, la herramienta utiliza las fuentes estándar instaladas en tu dispositivo para renderizar el texto.
