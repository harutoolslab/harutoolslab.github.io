Como usar a ferramenta de texto em imagens

Esta ferramenta permite adicionar texto a imagens de forma rápida e simples diretamente no seu navegador. Ideal para criar banners, adicionar anotações em redes sociais, ou inserir datas em fotos recordativas.

Principais Funcionalidades

  • Operação Intuitiva: Mova o texto adicionado arrastando-o com o rato ou com o dedo.
  • Múltiplos Textos: Pode colocar quantos blocos de texto desejar numa única imagem.
  • Privacidade Garantida: As imagens não são enviadas para o servidor. Todo o processamento é feito localmente no seu dispositivo, garantindo a segurança das suas fotos pessoais ou profissionais.

Perguntas Frequentes

P. Como posso apagar um texto?

R. Na versão atual, deverá recarregar a página ou carregar a imagem novamente para recomeçar. Planeamos adicionar a função de eliminação individual brevemente.

P. Posso usar fontes especiais?

R. Sim, a ferramenta utiliza as fontes padrão instaladas no seu dispositivo para a renderização.
