Como utilizar o Seletor de Cores

Esta ferramenta permite encontrar facilmente os "códigos de cores" necessários para web design ou criação de documentos. Basta clicar na paleta de cores central, escolher a sua cor favorita e o código pronto para usar no desenvolvimento será exibido.

Sobre os principais formatos

  • Formato HEX: Um código representado em hexadecimal como #ffffff. É o mais comum em desenvolvimento web (HTML/CSS).
  • Formato RGB: Representa a intensidade das três cores primárias (Vermelho, Verde, Azul) com valores de 0 a 255, como rgb(255, 255, 255). Comum em software de edição de imagem.

Perguntas Frequentes

P. Posso usar a função de conta-gotas para cores no ecrã?

R. Quando abre a paleta de cores, se o seu sistema operativo ou navegador tiver uma ferramenta de conta-gotas integrada, poderá utilizá-la para capturar cores de qualquer lugar no seu ecrã.

P. Posso obter formatos HSV ou CMYK?

R. Atualmente, suportamos apenas HEX e RGB focados em web design, mas planeamos adicionar formatos profissionais em futuras atualizações.
