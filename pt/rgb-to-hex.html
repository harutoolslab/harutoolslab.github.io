Como funciona a conversão de RGB para HEX

Existem várias formas de os computadores processarem cores, sendo o RGB e o HEX as mais representativas. No desenvolvimento web e na programação, é necessário alternar entre estes formatos, e esta ferramenta realiza essa transição instantaneamente.

O que é RGB?

Representa as três cores primárias da luz: "Red" (Vermelho), "Green" (Verde) e "Blue" (Azul), cada uma com uma intensidade de 0 a 255 (256 níveis). Se todos forem 0 (0, 0, 0), o resultado é "Preto"; se todos forem 255 (255, 255, 255), o resultado é "Branco".

O que é HEX (Código de Cor Hexadecimal)?

É um formato que converte cada valor RGB para base hexadecimal (0-9, a-f), organizando-os em pares de dois dígitos precedidos por "#" (ex: #ff0000). É o formato mais comum para especificar cores em HTML e CSS.

Método de Cálculo da Conversão

Divide-se o valor RGB por 16 e utiliza-se o quociente e o resto para obter os dígitos hexadecimais.

  • Exemplo: Se o R (Vermelho) for 255
  • 255 ÷ 16 = 15 com resto 15
  • Como 15 em hexadecimal é "f", o valor 255 torna-se "ff".

Como utilizar esta ferramenta

  1. Introduza os valores RGB: Insira números de 0 a 255 nos campos R, G e B.
  2. Verifique a cor: A cor do círculo de pré-visualização mudará de acordo com a sua entrada.
  3. Copie o HEX: Clique no botão de cópia por baixo do código que começa com "#" e cole-o no seu código CSS ou outro projeto.
