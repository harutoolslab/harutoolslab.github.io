Como utilizar a Lista de Códigos de Cores

Nesta página, reunimos os principais códigos de cores frequentemente utilizados em criação de websites, desenvolvimento de apps e trabalhos de design. Basta clicar em cada caixa de cor para que o código em formato HEX (#6 dígitos) seja copiado automaticamente para a sua área de transferência.

O que é um Código de Cor?

O código de cor (código hexadecimal) é um formato utilizado para representar cores em computadores. É expresso no formato "#RRGGBB", onde três elementos — Vermelho (Red), Verde (Green) e Azul (Blue) — são especificados em valores hexadecimais de 00 a FF, permitindo representar mais de 16 milhões de cores.

Perguntas Frequentes

P. Não consigo colar o código copiado.

R. O acesso à área de transferência pode estar restrito devido às definições de permissão do seu navegador. Verifique as permissões através do ícone de cadeado ao lado da barra de endereço.

P. Estão disponíveis formatos RGB ou RGBA?

R. Atualmente, mostramos apenas o formato HEX, que é o mais comum, mas planeamos adicionar uma função para alternar para o formato RGB em breve.
