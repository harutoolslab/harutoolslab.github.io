Como comprimir ficheiros PNG

O compressor de PNG do "HaruTools Lab" é uma ferramenta rápida de otimização que funciona diretamente no seu navegador. Embora os ficheiros PNG tendam a ser grandes, esta ferramenta permite reduzir o volume de dados até um nível que melhora significativamente a velocidade de carregamento de websites.

Características desta ferramenta

  • Compressão em lote: Pode arrastar várias imagens PNG de uma só vez para processamento simultâneo.
  • Sem instalação: Funciona inteiramente no navegador, sendo compatível com PC, Mac, Android e iOS.
  • Privacidade total: As imagens nunca são enviadas para o nosso servidor. Todo o processamento é concluído no seu próprio dispositivo.

Passo a passo

  1. Arraste e solte as imagens PNG que deseja comprimir na área indicada.
  2. Ajuste a "Taxa de compressão" no controlo deslizante (o valor 0.7 oferece um bom equilíbrio entre peso e qualidade).
  3. Clique no botão "Comprimir PNG".
  4. Verifique a pré-visualização e utilize o botão "Guardar" para descarregar a imagem otimizada.

Perguntas Frequentes (FAQ)

P. As transparências dos ficheiros PNG são mantidas?

A. Para alcançar uma taxa de compressão elevada, a lógica interna desta ferramenta utiliza a conversão para JPEG. Por este motivo, as áreas transparentes serão convertidas para branco ou preto. Se precisar de manter a transparência, recomendamos o uso de ferramentas específicas de redimensionamento.

P. A qualidade da imagem diminui muito?

A. Depende da configuração escolhida, mas com um ajuste de 0.8 ou superior, a redução do tamanho do ficheiro é feita com uma qualidade quase indistinguível a olho nu.
