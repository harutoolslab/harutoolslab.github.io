Política de Privacidade

O 【HaruTools Lab】 (doravante denominado "este site") estabelece a seguinte Política de Privacidade para o tratamento das informações pessoais dos usuários nos serviços oferecidos neste website (doravante denominados "o Serviço").

1. Tratamento de Dados (Processamento no Lado do Cliente)

A maioria das ferramentas disponibilizadas neste site, como edição de imagens, processamento de PDF, formatação de código e calculadoras, é executada utilizando JavaScript que roda diretamente no navegador do usuário.

  • As imagens enviadas ou os textos digitados pelos usuários não são transmitidos, salvos ou armazenados nos servidores deste site.
  • Como todo o processamento é realizado localmente no seu dispositivo, você pode utilizar a ferramenta com total segurança, mesmo para dados altamente confidenciais.

2. Distribuição de Anúncios

Este site utiliza o "Google AdSense", um serviço de publicidade de terceiros.

  • Fornecedores terceiros, incluindo o Google, utilizam cookies para veicular anúncios com base nas visitas anteriores do usuário a este ou a outros websites.
  • Com o uso de cookies de publicidade, o Google e os seus parceiros podem veicular anúncios para os usuários com base nas visitas efetuadas a este site e/ou a outros sites na Internet.
  • Os usuários podem optar por desativar a publicidade personalizada acessando as Configurações de Anúncios. Alternativamente, podem desativar o uso de cookies de fornecedores terceiros para publicidade personalizada visitando www.aboutads.info.

Para mais detalhes, consulte a página do Google sobre "Como o Google utiliza os dados quando o usuário usa sites ou aplicativos dos nossos parceiros".

3. Ferramentas de Análise de Acesso

Este site utiliza o "Google Analytics", uma ferramenta de análise de acesso fornecida pelo Google, para compreender o status de utilização do site e ajudar a melhorar o serviço. O Google Analytics utiliza cookies para coletar dados de tráfego. Esses dados de tráfego são coletados de forma anônima e não identificam o usuário individualmente.

4. Isenção de Responsabilidade

Não nos responsabilizamos por quaisquer danos ou prejuízos decorrentes do uso das informações ou ferramentas publicadas neste site. Embora tenhamos o máximo de cuidado no desenvolvimento em relação à precisão e segurança das ferramentas, não garantimos o seu conteúdo. Utilize o serviço sob sua própria responsabilidade.

Além disso, se você for redirecionado para outros sites por meio de links ou banners a partir deste site, não assumimos qualquer responsabilidade pelas informações, serviços, etc., fornecidos no site de destino.

5. Direitos Autorais e Direitos de Imagem

Os direitos autorais dos textos, imagens e programas das ferramentas contidos neste site pertencem ao operador deste site. É proibida a reprodução não autorizada ou a redistribuição secundária.

No entanto, em relação aos dados criados ou processados pelos usuários utilizando as ferramentas deste site, os usuários retêm todos os direitos e podem utilizá-los livremente para fins comerciais ou não comerciais.

6. Alterações na Política de Privacidade

Este site cumprirá as leis e regulamentações japonesas aplicáveis às informações pessoais e revisará periodicamente o conteúdo desta política para buscar sua melhoria.

A política de privacidade mais recente e revisada sempre será divulgada nesta página.
