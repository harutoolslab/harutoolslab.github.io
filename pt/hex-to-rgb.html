Sobre a conversão entre HEX e RGB

No mundo do desenvolvimento web e design gráfico, as cores são especificadas principalmente em "HEX (hexadecimal)" ou "RGB (decimal)". Com esta ferramenta, pode converter facilmente os códigos de cores que deseja usar no seu CSS e copiá-los diretamente para os seus projetos.

O que é HEX (Código de Cor Hexadecimal)?

Expressa cores através de 6 caracteres alfanuméricos após o símbolo "#" (ex: #FF5733). Os primeiros dois dígitos representam o Vermelho (R), os dois do meio o Verde (G) e os dois últimos o Azul (B). É o formato mais comum em HTML e CSS.

O que é RGB?

É um sistema que especifica as três cores primárias da luz — Red (Vermelho), Green (Verde) e Blue (Azul) — com valores entre 0 e 255 cada. É escrito como "rgb(255, 87, 51)", e o formato "rgba()", que adiciona transparência, também é usado frequentemente no web design.

Perguntas Frequentes (FAQ)

P. Posso converter se introduzir o código sem o símbolo "#"?

R. Sim, esta ferramenta deteta automaticamente a presença do "#", pelo que entradas como "FF0000" serão convertidas corretamente.

P. Suporta códigos HEX de 3 dígitos (ex: #F00)?

R. Sim, o código de 3 dígitos no formato abreviado é interpretado e convertido automaticamente para a versão de 6 dígitos.

P. É possível fazer o inverso, de RGB para HEX?

R. Atualmente, esta ferramenta é especializada na conversão de HEX para RGB, mas planeamos atualizações para expandir os campos de entrada em breve.
